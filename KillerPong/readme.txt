KillerPong game

Aliens attacking, closed square, you are trying to kill all 
enemies with paddle and ball. 
Some Killed enemies gives some things, like power ups, speed up for moving ball, or speed down (Can be more power ups, like bigger paddle).

During some time (90 seconds) you must score the biggest amount of points.

This is for the Power Up Table Tennis contest
(http://www.gamedev.net/topic/647849-power-up-table-tennis-contest/)
